# لوحة تحكم البائع - منصة إدارة المنتجات

**Al-Asli Dashboard** - منصة عربية RTL لإدارة المنتجات للبائعين غير التقنيين

## 📋 المميزات الرئيسية

✅ **واجهة عربية كاملة (RTL)** - كل النصوص والتخطيط معد للغة العربية  
✅ **لوحة تحكم سهلة الاستخدام** - إحصائيات وعرض سريع للمنتجات  
✅ **إضافة منتجات متقدمة** - بحث ذكي في 8,000+ منتج مع ملء تلقائي  
✅ **نموذج تفاعلي** - مع تحقق من البيانات (Zod + React Hook Form)  
✅ **رفع الصور** - معاينة فورية للصور المرفوعة  
✅ **متوافق مع الهاتف** - تصميم responsive للهواتف والأجهزة اللوحية  
✅ **رسائل تجربة المستخدم** - حالات التحميل والنجاح والأخطاء واضحة

## 🛠️ المكدس التقني

- **Next.js 14** (App Router) + TypeScript
- **Tailwind CSS** - للتصميم (مع دعم RTL كامل)
- **shadcn/ui** - المكونات UI
- **React Hook Form** - إدارة النماذج
- **Zod** - التحقق من البيانات
- **Lucide React** - الأيقونات

## 📖 دليل الاستخدام

### 🏠 الصفحة الرئيسية (لوحة التحكم)

**المسار:** `/`

المحتوى:
- **رؤوس الإحصائيات:**
  - إجمالي المنتجات
  - المنتجات النشطة
  - تمت إضافتها مؤخراً
- **إجراءات سريعة:**
  - زر "إضافة منتج جديد" (يقودك لصفحة الإضافة)
  - زر "عرض جميع المنتجات" (يقودك لقائمة المنتجات)
- **جدول المنتجات الأخيرة**

### ➕ صفحة إضافة منتج

**المسار:** `/add-product`

#### الحقول:

1. **بحث المنتج** (مطلوب)
   - Combobox قابل للبحث (fuzzy search)
   - يتصل ب `/api/click-products?q=`
   - يدعم آلاف المنتجات بكفاءة

2. **تفاصيل المنتج** (تملأ تلقائياً)
   - اسم المنتج
   - رقم المنتج
   - السعر
   - الكمية
   - المصنع

3. **الفئات** (مطلوبة)
   - Checkboxes متعددة الاختيار
   - الملابس، الأحذية، الاكسسوارات، الإلكترونيات، المنزل والحديقة

4. **وصف المنتج** (مطلوب)
   - Text area بـ 10-500 أحرف

5. **المقاسات** (مطلوب)
   - Radio buttons
   - خيارات: مقاس واحد، مقاس حر، S, M, L, XL, 2XL, 3XL

6. **الألوان** (مطلوب)
   - Checkboxes متعددة
   - 8 ألوان: أحمر، أزرق، أسود، أبيض، أخضر، أصفر، رمادي، بني

7. **صورة المنتج**
   - رفع الصور مع معاينة فورية

8. **حالة المنتج**
   - Toggle (نشط/غير نشط)

9. **زر الحفظ**
   - يحفظ البيانات إلى `/api/products`
   - رسائل حالة: جاري التحميل، نجاح، خطأ

### 📋 صفحة قائمة المنتجات

**المسار:** `/products`

- جدول بجميع المنتجات
- عرض: الاسم، الرقم، السعر، الكمية، الحالة

## 🔌 API Endpoints

### `GET /api/click-products?q={query}`

بحث عن المنتجات بـ fuzzy search

**المعاملات:**
- `q` (string): نص البحث

**الإجابة:**
```json
[
  {
    "value": "1",
    "label": "قميص أحمر (SHIRT-001)",
    "id": "1",
    "name": "قميص أحمر",
    "productNumber": "SHIRT-001",
    "price": 50,
    "quantity": 100,
    "manufacturer": "العلامة الأولى"
  }
]
```

### `POST /api/products`

حفظ منتج جديد

**الجسم:**
```json
{
  "productId": "1",
  "productName": "قميص أحمر",
  "productNumber": "SHIRT-001",
  "price": 50,
  "quantity": 100,
  "manufacturer": "العلامة الأولى",
  "categories": ["clothing"],
  "description": "قميص فاخر من الكتان الطبيعي",
  "sizeType": "m",
  "colors": ["red", "blue"],
  "image": "data:image/...",
  "isActive": true
}
```

**الإجابة:**
```json
{
  "success": true,
  "message": "تم حفظ المنتج بنجاح",
  "productId": "1712345678"
}
```

## 🎨 RTL والعربية

المشروع مُعد بالكامل للعربية:

- **Layout Direction:** كل الصفحات بـ `dir="rtl"`
- **Text Direction:** جميع النصوص محاذاة لليمين
- **Input Fields:** التوجيه الصحيح في حقول الإدخال
- **Tailwind:** classes معدلة للـ RTL

## 📱 الاستجابة للهاتف

جميع المكونات responsive:
- تصميم Grid متكيف (1 عمود على الهاتف، 2-3 أعمدة على الكمبيوتر)
- أزرار كبيرة الحجم (h-12, h-10)
- نصوص واضحة وكبيرة
- padding مناسب على الهاتف
